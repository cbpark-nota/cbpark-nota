Conversion 호환성 범위

LaunchX를 사용하여 AI 모델의 프레임워크를 목표 프레임워크로 자동 변환할 수 있습니다.

Conversion case

  1. ONNX to TensorRT

  2. ONNX to TensorFlow Lite

  3. ONNX to OpenVINO

  4. TensorFlow-Keras to TensorFlow Lite

호환 가능한 모델

업로드된 모델의 입력 레이어는 다음과 같아야 합니다.

  • 단일 입력 모델만 지원됩니다.

  • 이미지의 4차원 배열 구조는 배치, 채널 수, 높이, 너비 순으로 구성되어야 합니다.

  1. Batch size: 모델이 동시에 처리하는 결합된 입력 데이터셋의 수입니다.

  2. Channel: RGB 또는 BGR의 경우 3, 그레이스케일의 경우 1입니다.

  3. Input size: 컴퓨터 비전 작업에서 입력 크기는 입력 이미지의 크기를 의미합니다.
