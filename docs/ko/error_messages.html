Common Issues

Below is a list of common issues encountered while using LaunchX.

Upload

FrameworkException

업로드한 모델 파일이 지원하는 framework가 아닌 경우입니다. 지원범위 문서를 확인해주세요.

InputShapeLengthException

input shape 길이가 4가 아닌 경우입니다.

InspectFail

모델 파일에 문제가 있는 경우입니다.

KerasError

Keras 모델 파일에 문제가 있는 경우입니다.

MakeInputsError

Tensorflow 모델 파일에 서브그래프가 없는 경우입니다.

ModelFileInspectException

업로드된 모델 파일을 확인하는 과정에서 문제가 발생한 경우입니다.

ModelTypeError

모델 파일에 문제가 있는 경우입니다.

MultiLayerException

input layer 가 multi layer인 경우입니다.

OpenVINOError

OpenVINO 모델 파일에 문제가 있는 경우입니다.

TensorRTDeserializerError

TensorRT 모델 파일에 문제가 있는 경우입니다.

TooLargeBatchModelException

업로드하려는 모델의 batch size가 4를 초과한 경우입니다.

TooLargeChannelModelException

업로드하려는 모델의 channel이 3을 초과한 경우입니다.

Benchmark

BenchmarkFailedException

벤치마크 실행중 발생한 에러입니다. 상세한 원인을 파악하기 위하여 로그 조사가 필요합니다. [Contact support]

DynamicBatchModelException

upload된 model이 dynamic batch인 경우입니다. 현재 static batch 모델만 이용가능합니다.

DynamicInputShapeModelException

upload된 model이 dynamic width, height인 경우입니다. 현재 static Input size만 가능합니다.

StartBenchmarkFail

Device farm에서 벤치마크를 실행에 실패한 경우입니다. [Contact support]

Convert

ConvertFailedException

변환 실행중 발생한 에러입니다. 상세한 원인을 파악하기 위하여 로그 조사가 필요합니다. [Contact support]

NeedDeviceException

TensorRT로 변환을 하는 과정에서 변환을 하기 위한 device가 지정되지 않은 경우입니다. TensorRT는 device 위에서 변환을 하기 때문에 반드시 device와 Jetpack을 지정해야합니다.

NeedInputShapeException

dynamic shape을 가진 model을 변환하는 과정에서 model의 input shape 값이 없는 경우입니다.

Common cases

DeviceFarmUploadError

Device farm에서 model을 upload하는 과정에서 에러가 발생한 경우입니다. [Contact support]

DeviceNotMatchedWithFrameworkException

대상 프레임워크와 대상 기기가 일치하지 않는 경우입니다.

InternalException

일시적인 장애입니다. [Contact support]

LatestJobIsNotFinished

진행중인 변환이나 벤치마크 작업이 있는 상태에서 새로운 task를 실행하려고 한 경우입니다.

NoAvailableDevice

NetsPresso device farm에 이용 가능한 device가 없는 경우입니다.

NoCredit

작업을 실행하기 위한 충분한 크레딧이 없는 경우입니다. [Contact support]

NoModelUuidException

존재하지 않는 model_uuid 를 요청한 경우입니다.

NotSupportedDeviceException

입력된 target device가 선택한 framework에서 지원하는 device가 아닌 경우입니다.

AccountException

서버가 통신을 중계하는 과정에서 문제가 생기면 관련 메시지가 서비스 상에 표시됩니다. 정의되지 않은 예외 경우는 상세한 원인을 파악하기 위하여 로그 조사가 필요합니다. [Contact support]

Dataset file

Prepare dataset file for INT8 quantization with LaunchX

DatasetFileBrokenException

양자화 calibration을 위하여 업로드된 NumPy 파일을 불러오는데 실패한 경우입니다. NumPy 파일에 오류가 있을 가능성이 높습니다.

DatasetFileSuffixException

양자화 calibration을 위하여 업로드된 파일의 확장자가 .npy가 아닌 경우입니다.

InvalidTypeException

dataset file이 읽을 수 있는 NumPy array가 아닌 경우입니다.

NumpyArrayHasWrongShapeException

양자화 calibration을 위한 NumPy 파일의 shape이, 변환하려는 model의 input shape과 일치하지 않는 경우입니다.
